## Uso/Controles

Se puede utilizar joystick y flechas del teclado

W = Acelerar
A = Izquierda
S = Freno
D = Derecha

R = Reversa

Para poder dar marcha atras tiene que pulsarse R + W osea Poner Reversa y Acelerar
